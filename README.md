# Student Dormitory Information Management System
项目简介
该学生宿舍信息管理系统，采用前后端分离架构，双端完整功能，分为学生端，管理员端，并且具备完整业务流程。
技术栈
【后端】SpringBoot 3.1.5 + MyBatis 3.0.2 + MySQL 8.0
【前端】Vue 3.3.4 + Element Plus 2.4.2 + ECharts 5.4.3
【工具】JWT 4.4.0 + Pinia 2.1.7 + Axios 1.6.0
【构建】Maven + Vite 4.5.0
